A personal video studio built on HyperFrames — animated charts, product intros, AI-narrated TikTok hooks, PDF-to-video summaries, and competitor analysis.

View on GitHub What's HyperFrames?

What lives here

  • my-projects/ Each video lives in its own folder, with a HyperFrames composition and 15 AI skills auto-loaded for the coding agent.
  • prompts/ Reusable prompt templates for the workflows that come up most: CSV to chart, PDF to summary, TikTok hook, product intro, competitor analysis.
  • recipes/ Step-by-step workflows that have been proven. Each one points back to a real project that shipped using it.
  • brand/ Color palette, typography, motion principles. Drop the CSS variables into any composition for instant brand alignment.
  • skills/ Custom skills that work across every project — like a video analyzer for studying what competitors are doing.
  • memory.md Session log. Every meaningful work session adds an entry — what we did, what we learned, what's still open.

Set up on a new machine

Clone, restore the reference repos, add your API keys, and you're running.

git clone https://github.com/OchoOcho88/ocho-frames.git ~/Desktop/hyperframes
cd ~/Desktop/hyperframes
./setup.sh
cp .env.example .env  # fill in keys
cd my-projects/starter && npm install && npm run dev
