# Canva exports (drag-to-Canva staging)

Finished, Canva-ready assets land here so they're easy to find and drag into Canva.
The subfolders mirror the Canva folder structure.

## Structure (mirrors Canva)

- `IG-Ads/`  -> Canva folder **Sportif / IG Ads**  (https://www.canva.com/folder/FAHQK6gD-Do)
- `Source-Photos/` -> Canva folder **Sportif / Source Photos** (https://www.canva.com/folder/FAHQK2bvCGc)

Canva root folder **Sportif**: https://www.canva.com/folder/FAHQK6iSi6I

## How to transfer to Canva

1. Open the matching Canva folder in your browser (logged in).
2. Open this folder in Finder.
3. Drag the files onto the Canva folder page. They upload straight in.

(Direct programmatic upload is not possible: Canva's uploader only accepts public web
URLs, and we do not publish private brand assets to the internet.)

## Convention

- When we finish a Canva-ready asset, we copy it here (finished ads -> `IG-Ads/`,
  base photos -> `Source-Photos/`).
- This folder is **gitignored** (staging copies; the originals live in
  `clients/sportif/generated/images/` and are reproducible from the scripts).
